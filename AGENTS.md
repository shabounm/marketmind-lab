# MarketMind Lab Development Rules

Project: MarketMind Lab Trading Platform

## Project Structure
- WordPress + REHub Theme
- Custom development inside:
  - rehub-child
  - plugins/marketmind-core
  - plugins/marketmind-ai
  - assets/css

## Rules
- Never modify WordPress core
- Never modify original REHub theme
- Work only inside child theme and custom plugins
- Keep the design clean and professional
- Maintain RTL support for Arabic

## Design Style
- Dark premium finance theme
- Gold accents
- Clean trading dashboard layout

## Priority Development
1. Homepage layout
2. Market analysis pages
3. Trading Library (education articles)
4. Economic Calendar
5. MarketMind AI (members only)

## Goal
Build a premium Arabic market intelligence platform
with AI-powered analysis for forex, gold, and financial markets.
